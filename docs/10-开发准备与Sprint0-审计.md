当前 §7.1 "已完成操作"下包含 `git remote add origin`，但该操作未实际执行。建议在 Sprint 0 完成前执行如下操作：

### D.2 【建议】统一分支名

当前 `git branch` 返回 `master`，文档全篇引用 `main`。执行 `git branch -M main` 后即可消除差异。

`feat` 表示新功能特性，`chore` 表示项目基础设施。AdminAuthorizeAttribute 是基础设施代码（不包含业务规则），归为 `chore` 与 Sprint 0 "不写业务代码"的目标更一致。
