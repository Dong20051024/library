## 2. 关键技术决策

### 2.1 项目命名

| 项目 | 决策 | 依据 |
|------|------|------|

> **注意**：docs/07~10 原定项目名为 `LibrarySeatSystem.Web`，与用户要求的 `LibrarySeatReservation.Web` 不一致。本轮以用户要求为准，在 docs/12 中记录此差异。

### 2.2 架构选型

| 层 | 技术 | 版本 | 说明 |
|----|------|------|------|
| 运行时 | .NET | 8.0 | LTS 版本 |
| Web 框架 | ASP.NET Core MVC | 8.0 | Razor 视图引擎 |
| ORM | Entity Framework Core | 8.0.11 | Code-First |
| 数据库 | SQL Server LocalDB | 随 VS2022 | 本地开发 |
| 前端 | Bootstrap | 5.3.3 | CDN 引入 |
| 认证 | Session | — | 无 Identity |

### 2.3 路由方案

| 端 | 路由注册 | 示例 |
|----|----------|------|
| 用户端 | `MapControllerRoute("default")` | `/Seat/List` |
| 管理端 | `MapAreaControllerRoute("Admin")` | `/Admin/Reservation` |

管理端使用 Area 注册，Controller 放在 `Controllers/Admin/` 下，通过 `[Area("Admin")]` + `[Route("Admin/[controller]")]` 注册，视图放在 `Areas/Admin/Views/` 下。

### 2.4 认证方案

| 用户类型 | 方案 |
|----------|------|
| 学生（体验账号） | Session 存 `StudentName` 字符串，导航栏下拉切换 |
| 管理员 | Session 存 `AdminId` + `AdminDisplayName`，配合 `AdminAuthorizeAttribute` 过滤器 |

## 4. 数据库设计实现

### 4.1 表结构

| 表名 | 说明 | 记录数（种子） |
|------|------|---------------|
| Seat | 座位（7 字段） | 14 |
| Reservation | 预约记录（7 字段 + FK） | 6 |
| Admin | 管理员（4 字段 + 唯一索引） | 1 |

### 4.2 索引

| 表 | 索引名 | 字段 | 类型 |
|----|--------|------|------|
| Seat | UQ_Seat_Floor_Area_SeatNumber | (Floor, Area, SeatNumber) | 唯一索引 |
| Reservation | IX_Reservation_SeatId | SeatId | 普通索引 |
| Reservation | IX_Reservation_StudentName | StudentName | 普通索引 |
| Reservation | IX_Reservation_Status | Status | 普通索引 |
| Reservation | IX_Reservation_SeatId_TimeRange | (SeatId, StartTime, EndTime) | 复合索引 |
| Admin | UQ_Admin_Username | Username | 唯一索引 |

### 4.3 种子数据

| 数据 | 数量 | 说明 |
|------|------|------|
| 座位 | 14 | 3楼A区6个 + 3楼B区4个 + 5楼C区4个 |
| 其中空闲 | 11 | 可用 |
| 其中已预约 | 1 | 学生2预约 A-02，待签到 |
| 其中使用中 | 1 | 学生1使用 A-01 |
| 预约记录 | 6 | 2条使用中/待签到 + 2条已完成 + 2条已取消 |
| 管理员 | 1 | admin / 123456 |

## 7. 快速启动

详见 README.md，Sprint 0 完成后执行步骤：

```bash
# 1. 还原依赖
dotnet restore

# 2. 创建/更新数据库（种子数据自动插入）
dotnet ef database update

# 3. 运行
dotnet run

# 4. 访问
#    用户端: http://localhost:5000/
```

| 优先级 | 功能 | Controller | Service |
