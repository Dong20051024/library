```markdown
| **桌面端布局** | ...顶部是筛选条件行（状态筛选按钮组 + ~~搜索框~~ 搜索框¹）... |
```
或加脚注：
```
¹ 搜索框为前端客户端过滤（无需后端接口），仅在 UI 布局中保留，当前不连接后端搜索 API。
```
