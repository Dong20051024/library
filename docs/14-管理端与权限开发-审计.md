## E. 建议优化项

### E.1 【建议】补充座位启用/停用切换按钮

当前 `Admin/Seat` 页面的操作列只提供"编辑"和"删除"。缺少一个"启用/停用"按钮，使得禁用座位必须通过删除重建。

建议在操作列增加一个按钮，POST 到 `AdminSeatController.ToggleActive(int id)`：

```
// 在 SeatController 中新增
[HttpPost]
[ValidateAntiForgeryToken]
public IActionResult ToggleActive(int id) {
    _seatService.ToggleActive(id);
    return RedirectToAction(nameof(Index));
}
```

### E.2 【建议】管理端楼层/区域列表从后端传递

当前 Create/Edit Modal 中楼层（3楼/5楼）和区域（A区/B区/C区）是硬编码在 `<select>` 中的。建议在 `SeatService` 中增加 `GetFloors()` / `GetAreas()` 方法，通过 `ViewBag` 传到视图，保持与用户端 `Seat/List` 的筛选下拉一致。

### E.3 【建议】T02-10 + T01-11 浏览器端到端验收

两个验收卡累计到本轮仍未执行。功能代码已全部就位，建议在 Sprint 2b 或进入 Sprint 3 前集中完成一次完整的浏览器人工验收。
